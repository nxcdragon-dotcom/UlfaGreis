## üìù Common Tasks

### Add New News
1. Login to admin dashboard
2. Click "Kelola Berita" ‚Üí "Tambah Berita Baru"
3. Fill in: Judul (title), Konten (content), Upload Gambar (image)
4. Click "Simpan Berita" (Save)
5. Image will be saved to `/uploads/` folder

### Add New Official
1. Login to admin dashboard
2. Click "Kelola Pejabat" ‚Üí "Tambah Pejabat Baru"
3. Fill in: Nama (name), Posisi (position), Upload Foto (photo)
4. Click "Simpan Pejabat" (Save)
5. Photo will be saved to `/uploads/` folder

### Add Gallery Photo
1. Login to admin dashboard
2. Click "Kelola Galeri" ‚Üí "Tambah Foto Baru"
3. Fill in: Judul Foto (title), Upload Gambar (image)
4. Click "Simpan Foto" (Save)
5. Image will be saved to `/uploads/` folder

### Delete Content
1. Go to appropriate management page
2. Find item in the list
3. Click "üóëÔ∏è Hapus" (Delete) button
4. Content and associated image will be removed

## ‚ö†Ô∏è Important Notes

1. **Column Names are Case Sensitive**
   - Use `Judul` NOT `judul` for news table
   - Use `nama` for officials
   - Use `gambar` for image column (NOT `image`)

2. **Uploads Folder Must Exist**
   - `C:\laragon\www\desa_cendana\uploads\`
   - Folder permissions should be 755

3. **Password Storage**
   - All passwords are hashed with bcrypt
   - Cannot retrieve plain text passwords
   - To reset: Create new admin account or update in database

4. **Image Upload Limits**
   - Maximum file size: 5MB
   - Supported formats: JPG, PNG, GIF, WebP
   - File names are converted to timestamp-based names

5. **Session Management**
   - Sessions last for browser session duration
   - Login again if session expires
   - Logout button ends session immediately

## üêõ Troubleshooting

### "Unknown column" Error
- **Cause:** Column name mismatch between PHP code and database
- **Solution:** Check spelling of column names (case sensitive)
- **Status:** ‚úÖ FIXED (all files updated)

### File Upload Fails
- **Check:** Uploads folder exists and is writable
- **Command:** `chmod 755 uploads/` on Linux/Mac
- **Check:** File size under 5MB
- **Check:** File format is JPG, PNG, GIF, or WebP

### Cannot Login
- **Check:** Username and password correct
- **Check:** Admin account exists in users table
- **Check:** Database connection working

### Image Not Showing
- **Check:** File exists in `/uploads/` folder
- **Check:** Filename is correct in database
- **Check:** File permissions are readable (644)
